# Tamurfood

Private B2B instant ordering platform for a bakery in Bangladesh. Replaces phone calls with a simple web-based ordering system for shopkeepers.

See [Product Requirements Document](docs/prd.md) for full details.

## Tech Stack

- **Runtime:** Bun
- **Frontend:** React 19 + Vite
- **Routing:** TanStack Router
- **Data Fetching:** TanStack Query
- **UI:** Tailwind CSS + shadcn/ui
- **Backend:** Hono
- **Auth:** Better Auth
- **Database:** PostgreSQL + Drizzle ORM
- **Realtime:** Supabase Realtime
